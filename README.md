EventsApp
=========

Model
-----
Run `rake generate_erd` to regenerate (must have graphvis). Generated by Rails ERD.
![](/erd.png)

Ruby Version
------------
- ruby 2.1.5p273
- rails 4.1.8

System Dependencies
-------------------
Graphviz (only for Development Environment)

Database Creation
-----------------
```
$ rake db:migrate
$ rake db:seed
```
